# Memodown (Full stack)

## Description
An app which can add notiz or idea and share with your friend conveniently.

### Tool 
1. Next13 (TypeScript)
2. Mongoose
3. bcrypt
4. JWT(jsonwebtoken)
5. Style : Tailwind
6. State Management : Redux(RTK, API slice)

### Features & Discriptions
1. Authorization 
2. Rest API : CRUD maniupulation with MongoDB 
3. Send Message to friend
